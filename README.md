# D.planet

Este proyecto contiene un ejemplo sencillo de sitio web estático escrito en HTML, CSS y JavaScript.

## Estructura del proyecto

- `index.html` – página principal con secciones de inicio, sobre nosotros y contacto.
- `style.css` – estilos básicos para la página.
- `script.js` – script de ejemplo que muestra un mensaje en la consola.

Puedes abrir `index.html` en tu navegador para ver el resultado.
